文档说明

本文档是电商APP的服务器接口文档.

通用参数部分, 描述了统一的URL地址, 枚举参数和通用的JSON对象.

通用参数

基础URL

  • 接口基础URL: http://106.14.32.204/eshop/emobile/?url=
  • 品牌LOGO图片基础URL: http://106.14.32.204/eshop/data/brandlogo/

枚举参数

错误码

  1. 成功: 0
  2. 失败: 其它

排序方式

  1. 最贵: price_desc
  2. 最便宜: price_asc
  3. 最热: is_hot

规格类型

  1. 唯一: 0
  2. 单选: 1
  3. 多选: 2

通用JSON对象

响应状态

{   
    "succeed"    : 1,                 // 是否成功( 1 代表成功)
    "error_code" : 100,               // 错误代码
    "error_desc" : "Invalid session"  // 错误描述
}

轮播图

{
    "photo"      : {图片对象},         // 图片
    "description": "今天特价",         // 描述
    "url"        : ""                 // 外链URL
}

图片

{
    "small"      : "",                // 小图URL  
    "thumb"      : "",                // 中图URL
    "url"        : ""                 // 大图URL
}

简单商品

{
    "goods_id"      : 32,            // 商品ID
    "name"          : "标题",        // 商品名称
    "market_price"  : "商场价格",    // 商场价格
    "shop_price"    : "商店价格",    // 商店价格
    "promote_price" : "促销价格",    // 促销价格
    "brief"         : "简要",        // 简要
    "img"           : "{图片对象}"   // 图片
}

收藏的商品(继承简单商品的所有属性)

{
    "rec_id"        : 32             // 收藏id
}

商品分类

{
    "id"            : 1,              // 分类id
    "name"          : "鞋子",         // 分类名称
    "goods"         : [{简单商品对象}] // 推荐商品
}

过滤器

{
    "keywords"      : "关键词",          // 关键字
    "category_id"   : 100,              // 分类id
    "price_range"   : {价格区间},        // 价格区间
    "brand_id"      : 1,                // 品牌id
    "sort_by"       : "id_desc"         // 排序方式
}

价格区间

{
    "price_min": 390,                 // 最小价格
    "price_max": 400                  // 最大价格
}

分页参数

{
    "page"       : 1,               // 页码
    "count"      : 100                // 每页条数
}

分页结果

{
    "total"      : 100000,            // 总数
    "count"      : 100,               // 返回条数
    "more"       : 1                  // 是否还有更多 (1为是, 0为否)
}

首页商品浏览

轮播图和促销商品

基本信息

  • 请求类型: HTTP
  • 接口地址: /home/data
  • 请求方式: GET
  • 数据类型: X-WWW-FORM-URLENCODED
  • 响应类型: JSON

接口描述

获取首页的轮播图(首页主广告)和促销商品

响应数据

响应示例

{
    "status": {...}, // 参见响应状态对象
    "data": {
        "player": [
            {...} // 参见轮播图对象
        ], 
        "promote_goods": [
            {...} // 参见简单商品对象
        ]
    }
}
